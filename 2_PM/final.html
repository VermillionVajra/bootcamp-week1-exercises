This page is dedicated to brainstorming for CS50's Final Project!
I currently have a few ideas regarding creating linguistic tools that involve API which is a tool by which certain softwares can be adapted functionally to one's own program, at least in the case of Google Translate! Some ideas that I have include: creating a translation tool in the form of an IOS application; an adaptable conjugation chart for your choice of language, verb, or noun (if applicable); Finally, a morphology analyser for a few languages, such as
  1. Korean
  2. Japanese
  3. German
  4. Persian

*The project is now complete! It's called Akshara and it's a Sanskrit/Hindi alphabet instruction app! It functions in a few ways and has multiple interesting features:
  • Made using Swift and XCode
  • Uses a sliding flashcard feature that allows one to easily view information like an iOS gallery
  • Codically simple and can be modified as a mnemonic learning app or a simple viewing gallery
